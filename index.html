RETINA: Radio Echo Triangulation through Inter-Node Analysis

Existing FM radio transmissions act as illumination for low-cost passive rooftop radars. Long-range and accurate without powerful transmitters or FCC licenses needed. Connect to the network to triangulate radar hits from over 30km away.

Get in Touch See the Whitepaper

~$200 in Hardware to Setup

Radar nodes consist of: 1) a single board computer w. accessories 2) Two software defined radios 3) Two FM antennas

Open-source Radar Software

Nodes are preloaded with an OS and open-source radar package to record local objects' range and speed.

Servers Map Node Data

Nodes sends data to servers that plot radar tracks onto a global map. Data quality continually improves as overlapping nodes come online.
